# Image-Slider

A responsive and interactive **Marvel-themed image slider** built using **HTML**, **CSS**, and **JavaScript**. This project showcases dynamic sliding transitions and character-themed visuals perfect for fan pages, portfolios, or fun frontend practice!

## 🚀 Features

- 🔁 Smooth image transitions
- 🎨 Marvel-themed design
- 📱 Fully responsive layout
- ⚡ Lightweight and fast
- 💡 Beginner-friendly code structure

## 🛠️ Technologies Used

- **HTML5**
- **CSS3**
- **JavaScript**

## 📂 Folder Structure

main/ <br>
│ <br>
├── index.html  # Main HTML file <br>
├── style.css   # Styling for the slider <br>
├── script.js   # JavaScript for interactivity <br>

## 💡 Customization

- Adjust the style.css for custom themes or animations.
- Enhance functionality by adding auto-slide, navigation dots, or swipe support.
